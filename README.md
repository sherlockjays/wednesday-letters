매주 수요일, 낯선 누군가에게 편지를 쓰고 받는 익명 편지 교환 웹사이트입니다.

사용자는 자신의 일상과 감정을 편지로 작성하고, 운영자 검토를 거쳐 랜덤 매칭된 다른 사용자의 편지를 받아볼 수 있습니다. 이름도, 얼굴도 모르는 누군가와 진솔한 이야기를 나누며 위로와 연결을 경험하는 힐링 커뮤니티 플랫폼입니다.

## 주요 기능

- **편지 작성** — 매주 수요일마다 나의 일상과 감정을 담은 편지 작성
- **운영자 검토** — 건강한 커뮤니티를 위한 편지 검토 프로세스
- **랜덤 매칭** — 검토 완료된 편지를 다른 사용자와 익명으로 교환
- **익명성 보장** — 이름 없이 진솔하게 나누는 편지 경험

## 시작하기

```bash
# 의존성 설치
pnpm install

# 개발 서버 실행
pnpm dev
```
